# Library Management Software UML Diagram

The Library Management System is a software for the purpose of monitoring and controlling the transactions in a library. It studies on the library management system gives us the complete information about the library and the daily transactions done in a Library. For maintain the record and retrieve the details of books in the library which mainly focuses on basic operations in a library like adding new member, new books, new information, searching books, members, facility to borrow and return books.
